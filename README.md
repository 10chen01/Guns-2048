# Guns-2048
一个2048的变体，用枪械名字取代传统2048的数字，更有乐趣。

比如M4A1,HK416,HK417,AK47之类著名枪械种类繁多。

努力取得核弹，结束游戏

枪械&2048迷们看过来!

## 加倍奖励

除以上普通块以外，我们还特殊做了一种块。

所有普通块都能和加强块合并，并依照加强块的面值进行乘法操作

加强块合并形成更强的加强块
