# Guns-2048
一个2048的变体，用枪械名字取代传统2048的数字，更有乐趣。

比如M4A1,HK416,HK417,AK47之类著名枪械种类繁多。

努力取得核弹，结束游戏

枪械&2048迷们看过来!
